handwritten-code-pipeline Repo pending release

A prototype system for university courses that lets students submit photos of handwritten C++ code. A locally run LLM (via Ollama) performs OCR to translate the handwriting into source code, the student reviews and corrects the translation, and the code is compiled and executed inside a secure, sandboxed container. Results and session history are stored server-side and exportable as a PDF for assignment submission.

Tech: Python, Flask, Ollama, HTML, JavaScript, Podman, SQLite

This is an active research project. The public repo will be released once the current working phase is complete — check back on my GitHub for updates.
